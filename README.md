# VocabQuiz——你的专属轻量单词复习小站

## 项目概述

英语单词Quiz是一个纯前端的英语单词学习工具，专注于帮助用户高效记忆和复习英语单词。该应用采用现代化的Web技术，提供了丰富的功能和良好的用户体验，支持离线使用，无需后端服务器。

### 核心价值

- **离线使用**：纯前端实现，无需网络连接
- **多词库管理**：支持创建和管理多个词库
- **灵活导入**：支持文本、JSON文件和PDF文件导入
- **智能复习**：自动标记需要复习的单词
- **实时进度**：可视化学习进度和统计数据
- **响应式设计**：适配不同设备屏幕

## 核心功能模块

### 1. 单词测试

- 随机显示单词，隐藏中文翻译
- 支持「认识，下一个」、「查看中文」、「不认识，标记复习」、「跳过」等操作
- 实时显示学习进度和统计数据
- 测试完成后显示结果

### 2. 词库管理

- 创建、编辑、删除词库
- 查看词库详细信息（单词数量、创建时间等）
- 切换不同词库进行测试

### 3. 单词导入

- 支持文本格式导入（英文,中文）
- 支持JSON文件导入
- 支持加载默认单词
- 支持从PDF文件提取单词（需要先运行extract\_words.py）

### 4. 复习系统

- 集中显示标记为「不认识」的单词
- 提供专门的复习模式
- 支持清除复习列表

### 5. 数据存储

- 使用IndexedDB存储词库和单词数据
- 使用localStorage缓存测试进度和复习列表
- 确保数据持久化，刷新页面后数据不丢失

## 关键技术详解

### 1. IndexedDB

- **用途**：存储词库和单词数据
- **优势**：支持大量数据存储，异步操作，事务支持
- **实现**：使用原生IndexedDB API，创建了libraries和words两个对象存储

### 2. localStorage

- **用途**：缓存测试进度和复习列表
- **优势**：简单易用，适合存储小量数据
- **实现**：存储测试状态、复习列表和已知单词数量

### 3. 响应式设计

- **实现**：使用CSS媒体查询和弹性布局
- **效果**：在桌面、平板和手机上都能良好显示

### 4. PDF单词提取

- **工具**：Python的pdfplumber库
- **实现**：使用正则表达式匹配PDF中的单词格式
- **输出**：生成标准的JSON格式单词数据

## 性能优化

### 1. 前端性能

- **IndexedDB批量操作**：使用事务批量处理单词数据
- **DOM操作优化**：减少重排和重绘
- **事件委托**：减少事件监听器数量

### 2. 数据处理

- **批量导入**：支持一次性导入大量单词
- **JSON解析**：高效处理JSON格式数据
- **PDF提取**：使用正则表达式快速匹配单词

### 3. 用户体验

- **实时反馈**：操作后立即显示Toast提示
- **进度更新**：实时更新学习进度条
- **动画效果**：平滑的过渡动画

## 使用场景示例

### 场景1：日常单词学习

1. 创建一个新的词库，命名为「日常词汇」
2. 导入相关单词
3. 开始测试，标记不认识的单词
4. 在复习页面集中复习标记的单词

### 场景2：考试备考

1. 从PDF教材中提取单词（运行extract\_words.py）
2. 将提取的单词导入到新建的「考试词汇」词库
3. 系统测试，重点复习标记的单词
4. 跟踪学习进度，确保覆盖所有单词

### 场景3：自定义词库

1. 按主题创建多个词库（如商务英语、旅游英语等）
2. 为每个词库导入相关单词
3. 切换不同词库进行针对性学习
4. 根据学习情况调整词库内容

## 安装部署指南

### 前端应用

1. **克隆项目**：`git clone <项目地址>`
2. **直接打开**：在浏览器中打开 `index.html` 文件
3. **离线使用**：可以将整个项目文件夹保存到本地，无需服务器

### PDF单词提取工具

1. **安装依赖**：`pip install pdfplumber`
2. **准备PDF**：将PDF文件放在项目根目录
3. **运行脚本**：`python extract_words.py`
4. **查看结果**：生成的单词数据会保存到 `words.json` 文件
