# VocabQuiz - 英语单词测试应用

## 项目概述

VocabQuiz是一个现代化的英语单词测试应用，旨在帮助用户更有效地学习和记忆英语单词。应用采用纯前端技术栈，无需后端服务器，所有数据存储在浏览器本地，确保用户数据的隐私和安全。

## 核心功能

### 🎯 单词测试
- 随机显示英文单词，中文翻译默认隐藏
- 支持"认识，下一个"和"不认识，标记复习"操作
- 实时显示学习进度和统计数据
- 支持上一个单词回溯功能

### 📚 词库管理
- 创建和管理多个词库
- 导入单词列表（支持文本和JSON格式）
- 导出词库为JSON文件
- 内置默认词库

### 🔄 复习模式
- 集中复习标记的单词
- 复习模式下不认识的单词会自动移到列表末尾
- 复习完成后可直接切换回测试模式

### 📊 学习统计
- 实时学习进度图表
- 单词掌握情况分析
- 学习历史记录
- 平均准确率和学习时间统计

### 🔊 单词发音
- 使用Web Speech API实现单词发音
- 支持点击按钮听单词发音

### 🎨 现代UI设计
- 响应式布局，支持桌面端和移动端
- 流畅的动画和微交互效果
- 清晰的视觉层次和色彩方案
- 直观的用户界面

## 技术架构

### 技术栈
- **前端框架**: 纯HTML5 + CSS3 + JavaScript
- **构建工具**: Vite
- **数据可视化**: Chart.js
- **本地存储**: IndexedDB + localStorage
- **代码规范**: ESLint

## 安装部署

### 开发环境
1. 克隆项目到本地
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 访问：`http://localhost:3000`

### 生产部署
1. 构建生产版本：`npm run build`
2. 将 `dist` 目录下的文件部署到任何静态文件服务器
3. 访问部署后的地址

## 使用指南

### 首次使用
1. 打开应用后，系统会自动创建默认词库并加载默认单词
2. 点击"开始测试"按钮开始测试
3. 对于认识的单词，点击"认识，下一个"
4. 对于不认识的单词，点击"查看中文"后点击"不认识，标记复习"

### 词库管理
1. 点击"词库管理"导航栏
2. 点击"新建词库"按钮创建新的词库
3. 点击词库的"使用"按钮切换到该词库
4. 点击"删除"按钮删除词库

### 导入单词
1. 点击"导入单词"导航栏
2. 选择要导入的词库
3. 输入单词列表（格式：英文,中文 或 JSON格式）
4. 点击"导入文本"按钮导入单词
5. 或点击"从本地JSON文件导入"按钮导入JSON文件

### 复习单词
1. 点击"我的复习"导航栏
2. 点击"开始复习"按钮开始复习标记的单词
3. 复习完成后可点击"切换模式"回到测试模式

### 学习统计
1. 在测试页面查看实时学习进度图表
2. 查看单词掌握情况分析
3. 系统会自动记录学习历史和统计数据

## 性能优化

- **代码分割**: 使用Vite的代码分割功能，减小初始加载体积
- **懒加载**: 动态导入非核心模块，如可视化和统计功能
- **缓存策略**: 合理使用localStorage缓存数据
- **DOM操作优化**: 减少DOM操作，使用事件委托
- **动画性能**: 使用CSS transform和opacity实现流畅动画

## 安全性

- **本地存储**: 所有数据存储在浏览器本地，无需网络请求
- **输入验证**: 对用户输入进行验证，防止XSS攻击
- **数据安全**: 敏感操作需要用户确认

## 浏览器兼容性

- 现代浏览器：Chrome, Firefox, Safari, Edge
- 支持Web Speech API的浏览器可使用单词发音功能
